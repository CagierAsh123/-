# 延边朝鲜族自治州 · 数据可视化地图

> 吉林省延边朝鲜族自治州景点数据可视化交互地图，基于 React + Vite + Tailwind CSS + Framer Motion 构建。

## 两种使用方式

本项目有两套独立的打开方式，适用于不同场景：

### `start.bat` — 编辑模式（需要 Node.js）

双击 `start.bat`，首次自动安装依赖，之后自动打开浏览器。

这是完整的开发环境，可以使用所有编辑工具（画笔、标点、编辑、调试等）。修改会实时保存在浏览器 localStorage 中。

编辑完成后，有两种方式把内容交给别人：

**方式 A：导出 JSON（推荐）**
1. 点底栏「导出数据」→ 浏览器下载一个 `.json` 文件
2. 把 JSON 文件发给对方
3. 对方用 `open.bat` 打开地图后，点「导入数据」加载 JSON 即可

**方式 B：烧录进构建**
1. 用 `start.bat` 编辑好内容
2. 双击 `build.bat`（或 `npm run build`）
3. 构建时会把当前 `src/data/yanbian/userData.js` 中的数据打包进 `dist/`
4. 把整个 `dist/` 文件夹发给对方，对方双击 `open.bat` 即可看到

### `dist/open.bat` — 查看模式（零依赖）

双击 `dist/open.bat` 即可打开地图。不需要 Node.js，不需要任何编程环境（利用 Windows 自带的 PowerShell 启动本地服务器）。

这是只读的静态版本，但支持通过「导入数据」按钮加载 JSON 文件来更新内容。

**总结：`start.bat` 负责编辑，`open.bat` 负责查看。JSON 文件是两者之间的数据桥梁。**

## 地图交互

- 滚轮缩放（最大 6 倍），拖拽平移，双击重置视图
- 点击景点弹出信息名片，支持 中/英/日/俄/韩 五语切换

## 底栏工具

| 按钮 | 功能 |
|------|------|
| 经纬网 | 开关经纬度网格（每 0.5° 一条线，整度标注） |
| 画笔 | Emoji 画笔工具，自由装饰地图（山川河流森林动物等），支持分类选择、自定义输入、大小调节、橡皮擦、撤销 |
| 标点 | 自定义标点工具，添加带蓝色呼吸光圈的交互式地标，支持 5 语言信息填写 |
| 编辑 | 手动调整预设景点的 SVG 位置 |
| 调试 | 拖动锚点校准地理投影参数 |
| 设置 | 画面调节（亮度/对比度/饱和度/色相/光晕强度） |
| 导出数据 | 下载 JSON 文件，包含所有画笔 emoji + 自定义标点 + 画面设置 |
| 导入数据 | 从 JSON 文件恢复所有自定义内容 |

## 关于工期与实现说明

本项目作为延边州数据可视化地图的快速原型，在约一天时间内完成从零到可交互的全流程开发。考虑到工期限制，采取了以下策略：

- 内置自定义工具（画笔/标点/编辑）— 可自行添加景点数据、装饰地图元素，无需修改代码
- 多语言模板化 — 填入各语言标题和描述即可，名片自动适配
- JSON 导入导出 — 编辑成果可跨浏览器、跨设备传递

我看过了实例视频，我的结论是要达到那种程度工期需要按星期计算，而且对方大概是团队项目，而非纯粹的 vibe coding 结果，其中的美术、数据来源等我无法企及。差不多意思到了，后续如需进一步完善可以在此基础上迭代。

## 预设景点（11 处）

六鼎山文化旅游区（敦化）、珲春防川景区（珲春）、帽儿山国家森林公园（延吉）、中国朝鲜族民俗园（延吉）、长白山景区北坡（安图）、琵岩山风景区（龙井）、老里克湖景区（和龙/安图）、雁鸣湖丹江湿地公园（敦化）、图们口岸景区（图们）、延边博物馆（延吉）、金达莱民俗村（和龙）

## 技术栈

React 19 + Vite 8 + Tailwind CSS 4 + Framer Motion，SVG 矢量地图（8 县市 + 外边界），主组件 `YanbianMap.jsx` 包含全部交互逻辑。
